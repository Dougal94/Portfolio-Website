Password Generator

This password generator was created purely with Javascript. The example prompt given by Codecademy was to create a message generator that created three seperate statements with random words filling blanks each time. I personally wasn't interested in creating messages, so I created a code block that takes a number input for the length of a passcode and generates a random sequence of letters (both capital and lower case), numbers and punctuation of the chosen length.

    Languages Used:

  • Javascript
